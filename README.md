
# Patient Folder Work Sample

A mock patient records folder structure demonstrating correct
medical imaging file naming conventions.

## Naming convention
`YYYYMMDD_CaseID_Procedure_View_Stage`

## Cases included

### C00123 — Adult Male, 47
| Date | Procedure | View | Stage |
|------|-----------|------|-------|
| 20240315 | XRAY | AP, LAT | PRE |
| 20240316 | FLUORO | AP, LAT | INTRA |
| 20240318 | XRAY | AP, LAT | POST |

### C00124 — Adult Female, 62
| Date | Procedure | View | Stage |
|------|-----------|------|-------|
| 20240320 | MRI | SAG | PRE |
| 20240322 | CT | AX | POST |

## Modalities
XRAY · Fluoroscopy · MRI · CT

## Stages
`PRE` = pre-procedure · `INTRA` = during · `POST` = post-procedure

## Purpose
Portfolio piece demonstrating understanding of clinical imaging
file management and medical record-keeping standards.
